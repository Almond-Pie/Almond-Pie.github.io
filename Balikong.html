01 — Brief

A name that already
tells the story

A balisong trainer is a butterfly knife with no sharp edge — a practice tool for learning the flipping tricks that make balisong culture what it is. The audience is niche, knowledgeable, and has strong opinions about quality. They are not looking for a generic sports-equipment brand.

The name Balikong is the whole brief compressed. Bali from balisong, Kong from the gorilla — the mascot that embodies the raw, slightly dangerous energy of the product without taking itself too seriously. The name is a pun and a character in two words.

"The name does the positioning before the logo even loads."

02 — Insight

Bold mascots
earn loyalty

Balisong culture lives on YouTube, Reddit, and Discord — communities built around skill and personality. Brands that resonate here don't look corporate. They look like something a person chose, not a committee approved.

A gorilla — confrontational, heavyweight, holding the product — communicates everything: respect the craft, this isn't a toy, but we're not precious about it either. The tan and near-black palette keeps it tactile, like silk-screened on a garment rather than printed on a brochure.

03 — Mark

Three rounds to
the right blade

The logo went through three distinct passes. Each round tested a different approach to the balisong in the gorilla's hand — the single most important detail, because the knife IS the product category, and it has to read correctly at small sizes.

V1
Closed balisong — technically accurate, reads as any knife at small size
V2
Open crescent blade — better silhouette, gorilla illustration refined
V3 — Final
Cleanest linework, strongest silhouette, blade reads at all sizes

04 — Web

Storefront built
in React + TypeScript

Alongside the brand, the project includes a full storefront scaffolded in React 18, TypeScript, and Vite. The architecture is component-first: each page and UI piece is isolated, making it easy to iterate on the shop experience without touching the surrounding layout.

src/
├─ pages/
│ ├─ Home.tsx — hero + featured products
│ ├─ Shop.tsx — product grid with filters
│ ├─ About.tsx — brand story
│ └─ Contact.tsx — enquiries
├─ components/
│ ├─ Header.tsx — nav + logo
│ ├─ Footer.tsx — links + legal
│ ├─ ProductCard.tsx — image, name, price, CTA
│ └─ Cart.tsx — basket state
└─ utils/
├─ api.ts — product data layer
└─ helpers.ts — formatting, currency
Frontend

React 18 + TypeScript

Typed throughout. React Router v6 handles client-side navigation between Home, Shop, About, and Contact with no page reloads.

Tooling

Vite

Sub-second HMR during development. Production builds are tree-shaken and output-optimised — the storefront stays fast as the product catalogue grows.

Components

ProductCard

Reusable card component takes image, name, and price as props — product data can be swapped to a real API without touching the UI layer.

Design system

Brand tokens in CSS

Kong Tan and Jungle Dark are defined as CSS custom properties in theme.css, so every component inherits the palette from a single source of truth.
